Calculdora de Gastos de Eletronicos.
Explicação da Utilização!

Introdução

Essa RestAPI aceita cadastro de usuarios e eletronicos e calcula os consumos
aproximados com base nas taxas da CPFL na região de Ribeirão Preto, desconsiderando
outras taxas como outras bandeiras além da verde e Contribuição Custeio IP-CIP, taxas
essas que são além do custo por kwh.

Operações:

Todos os servicos poder ser consumidos a partir do seguinte contexto:
../consumerAPI/webresources/consumer/..

Cadastro de usuário:
../consumerAPI/webresources/consumer/cadastrar
POST
submissão :
{
"name": "joao",
"email": "joao@gmail.com",
"password": "pass3",
}

Login de usuário:
../consumerAPI/webresources/consumer/login
POST
submissão :
{

"email": "joao@gmail.com",
"password": "pass3",
}
respota :
{
"id": 7,
"name": "tese1",
"email": "teste@gmail.com",
"password": ""
}


Usa-se o id do usuário para cadastrar os eletronicos:
../consumerAPI/webresources/consumer/cadastrareletro
POST
{
"name": "notebook",
"marca": "dell",
"potencia": 90,
"tempoUso": 8.00,
"pessoaIdFk": 2
}

Busca os equipamentos pelo id do usuário com com os cálculos:
../consumerAPI/webresources/consumer/buscaeletro/{id}

Editar as informações dos eletronicos:
../consumerAPI/webresources/consumer/editaeletro
PUT
Submissão :
[
{
"id": 1,
"name": "notebook",
"marca": "dell",
"potencia": 90,
"tempoUso": 8.00,
"pessoaIdFk":2
}

Trocar a senha do usuário:
../consumerAPI/webresources/consumer/trocasenha
PUT
Submissão :
{
"name": "joao",
"email": "joao@gmail.com",
"password": "pass4",
"passwordtwo" : "pass5"
}

Deletar os equipamentos pelo id:
../consumerAPI/webresources/consumer/deletaeletro/{id}
DELETE

Deletar os usuário pelo id:
../consumerAPI/webresources/consumer/deletauser/{id}
DELETE
